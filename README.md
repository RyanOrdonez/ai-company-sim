# AI Company Simulator

A gamified, browser-based AI company simulator where you hire AI agents with real personalities, assign real work, and build a company in a stylized 3D office.

**Status:** In Development — Phase 1 (Foundation)

## What Is This?

You are the CEO. You build an office. You interview and hire AI agents who have names, faces, personalities, and opinions. You assign them real tasks — content writing, code generation, strategy briefs, market research — and they produce real deliverables you can download and use.

The game is the work. The work is the game.

## Tech Stack

- **Frontend:** React + Vite + TypeScript
- **3D Engine:** Babylon.js
- **Backend:** Node.js (Fastify) + TypeScript
- **Database:** Supabase (PostgreSQL)
- **AI:** Claude API (primary) + multi-provider fallback
- **Queue:** BullMQ + Redis
